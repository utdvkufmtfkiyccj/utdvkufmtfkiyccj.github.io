STANDARD Drum Bag Set for Club-JAM Kit

バッグ3枚がセットになったドラムバッグセット。特にClub-JAMシリーズの運搬に最適で、3枚のバッグに10インチ・タム、13インチ・スネアと14インチ・フロア、そして18インチ・バスドラムを収納可能です。持ち手ハンドルとストラップ付（※注意：18インチ用はハンドルのみ）。軽量なため、ポータビリティに優れています。

セット内容

(1)18インチ・バスドラム用 x 1

(2)10インチ・タム用 x 1

(3)14インチ・フロア + 13インチ・スネアドラム用 x 1

主な特長・仕様

○全面に10mm厚のクッションと600デニールの生地を採用

○14インチ用のバッグには、シェルが干渉しないような中敷きが付属しています。

○14インチ用と10インチ用はハンドルとストラップ付（18インチ用はハンドルのみ）

○内寸サイズ：10インチ/φ330mm x 230mm、14インチ/φ415mm x 340mm、18インチ/φ540mm x 360mm(※全てメーカーでの実測値です)
