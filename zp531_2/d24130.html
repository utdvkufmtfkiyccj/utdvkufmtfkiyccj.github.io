【商品概要】
USB Type-C機器専用の UCB-C ドッキングステーション 12in-1です。USB-C から HDMI / VGA / USB 3.1 Gen 1-C / USB 3.0x3 / RJ45 / SD / TF / TRRS 3.5mm / PD / DC に接続可能です。12in1で様々な用途に役立ちます。（ポート毎の内容は下記商品コンテンツページをご確認ください）
USB3.1 Gen 1-Cポート：USB-Cハードドライブ、USBフラッシュドライブ、カードリーダー、データの読み込み/書き込みに対応
USB-C充電ポート：ノートパソコンの安定した電源供給を可能にする高速充電です。MacBook Pro13のフル充電に約1.5時間 、MateBook13のフル充電に約2時間
3つのUSBポート：マウス、キーボード、フラッシュディスクなどに接続できる3つのUSBポート搭載 理論速度5Gbpsで1GBファイルを5秒で伝送
高速データ転送：USB-AポートとUSB-Cポートはそれぞれ最大5Gbpsの高速データ転送に対応し、HD動画も数秒で転送可能。SD/TFカードスロットを搭載されたUSB ドッキングステーションはSDとMicro SDが同時に読み書き可能で、SD、SDHC、Micro SD、MMC、SDXCなどのメモリカードに対応します。
