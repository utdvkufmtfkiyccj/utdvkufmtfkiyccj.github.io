- Features -
Function 01
表情

いくつもの三角形のガラスを交互に隣合わせて多角形に仕上げています。微妙な大きさの違いや溶接部の掠れなど、ハンドメイドならではの素朴なテイストをお楽しみください。
   
Function 02
角度

アンティークゴールドのソケットがレトロなムードを漂わせます。ソケットの角度を変えて、お好きな方向に光を届けることができます。
   
Function 03
OFF時も

消灯時にはガラスの質感や透明感をより味わえます。ON/OFFで印象が変わる趣のあるデザインです。
 
 
Function 04
サイズ感

デザインを損なわず実用性を両立させた、ベストなサイズ感を追求。

- Brand information -
 
「人が過ごす場所と時間を心地よく」

インターフォルムは1979年に照明器具のメーカーとして創業して以来、時計やインテリア雑貨にもフィールドを広げ、みなさまの日々の暮らしのひとときを心地よくするお手伝いをしたいという想いを込めて、オリジナルデザインの商品をご提案してきました。

インターフォルムという社名には、「世界に通用するデザインを」という意味が込められています。
そんな理想を持ち、世界のトレンドと、日本の緻密なデザインワークをかけ合わせた商品づくりを続けています。
