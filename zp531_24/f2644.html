    トランスミッターとコードを照合する方法:
    1. トランスミッターとレシーバーを30~50cm以内に近づけます。
    2. 送信機の電源を入れ、R4FGMの電源を入れます。
    3. R4FGMの側面に黒いボタンがあり、ボタンを1秒間長押しして放すと、レシーバーのライトが点滅し始め、バインドが正常に終了すると常に点灯します。

    動作モード
    通常動作モード -- 緑色のLED、ジャイロは動作しません。
    ジャイロ機能動作モード -- 緑と赤の両方のLEDが点灯し、赤色のLEDが点灯します。
    ID SET スイッチを 1 秒以内に 3 回クイック押します。LEDが緑から赤に変化するとジャイロ機能がオンになります。

    ジャイロ機能をオンにする方法
    ID SET スイッチを 1 秒以内に 3 回クイック押します。LED は緑から赤に変化し、ジャイロ機能がオンであることを意味します。

    ジャイロリバース
    飛行機が逆転しているのと同様、RCカー用のジャイロも逆転しています。 逆キャリブレーションには、正しいジャイロリバース設定が必要です。
