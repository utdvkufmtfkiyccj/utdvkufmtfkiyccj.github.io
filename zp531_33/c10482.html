  • TCS34725FN is a low-cost, cost-effective RGB full-color color recognition sensor. The sensor recognizes the surface color of an object through optical sensing. It supports the three primary colors of red, green, and blue (RGB), supports bright light sensing, and can output the corresponding specific values to help you restore the true colors, and includes an IR blocking filter making it an ideal color sensing solution for use under varied lighting conditions.
  • CQRobot RGB Color SensorOnboard TCS34725FN, embedded ADC, high sensitivity, wide dynamic range. I2C communication, uses a few pins. Integrates IR blocking filter, minimizes the IR spectral component of the incoming light.
  • Outputs RGB data without white balance. Outputs light intensity, feels the light like human eyes. Supports light intensity interrupt output, programmable upper and lower thresholds. Supports fill-light by onboard LED, adjustable brightness by PWM.
  • Onboard voltage level translator, compatible with 3.3V/5V operating voltage. Compatible with motherboards such as Raspberry Pi/Arduino/STM32.

説明:

This is a color sensor that supports RGB color data and light intensity output via the I2C interface. The color sensor is based on TCS34725FN color optical digital converter, it output red, green, blue (RGB) clear optical sensor value of digital. The integrated infrared blocking filter minimizes the infrared spectral of incident light , the sensor can accurately measures color. it has high sensitivity, wide dynamic range. Minimize IR and UV spectral component for accurate color measurements. It also has ambient light intensity detection and can be shielded interrupt.
